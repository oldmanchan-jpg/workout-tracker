## Standard Work Report (Required in every Cursor response)
After completing any task, output a section titled "WORK REPORT" with:
1) Summary (1–3 bullets)
2) Files changed (bulleted list with paths)
3) For each file changed: what changed (1–2 bullets)
4) Commands run (exact commands, or "none")
5) Verification (state whether any app code was changed; if yes, list files)
6) Blockers / open questions
7) Next suggested task (one sentence, optional)
